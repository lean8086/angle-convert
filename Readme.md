
# angle-convert

  Convert radians to degrees and back

## Installation

  Install with [component(1)](http://component.io):

    $ component install llinares/angle-convert

## API

### degToRad(deg)

```js
degToRad(128.34); // Returns 2.2399555620095226
```

### radToDeg(rad)

```js
radToDeg(2.24); // Returns 128.3425461093044
```
